# Competitive Programming Checklist

Live page:
https://cali042008.github.io/competitive-programming-checklist/

## Description

This repository hosts an interactive competitive programming checklist designed for progressive practice and motivation.

The site includes:

- roadmap-ordered chapters
- problem-level checkboxes with saved progress
- progress bar and completion metrics
- filters by source (`OJI`, `ONI`, `Lot`, `Baraj`, `Codeforces`, `Other`)
- filters by class level from the official olympiad program
- search across chapters, topics, and problems
